# RN Auth - Demo

### Requirements

```
Node 24
XCode/Android Studio
```

### Installation

```
bun install
bun start
run on your device
```

### Features Integrated

- [x] Auth Screen with Validation (I did put this as one screen with a separate identifier since I can reuse more code that way.)
  - [x] Login
  - [x] Sign Up
- [x] Homepage showing user name and user details
- [x] Persisted Auth & Auth Context
- [x] Navigation between Sign Up, Log In, Home Screen
- [x] UI - Drawer and Drawer Log out (Don't worry I still kept the button on the home page as per requirement)
